# Replace Values with NA

Replaces specified values in a vector with `NA`.

## Usage

``` r
naIf(x, values)
```

## Arguments

- x:

  A vector.

- values:

  Values to be replaced by `NA`.

## Value

A vector of the same type as `x`.
